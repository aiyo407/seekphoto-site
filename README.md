- 手机、相机、网盘备份的照片越积越多，想找一张图却要翻半天。

## 核心功能

### 🔍 文字找图
输入一句话，比如“海边的日落”“宝宝的笑容”“穿红衣服的人”，AI 即刻从海量照片中返回最匹配的结果。

- 支持中文自然语言搜索
- 也能用一张照片搜索相似照片
- 不用记住文件名，不用一张张翻看

### 👤 人物分类
自动识别照片中的人脸，把同一个人的照片归为一组。家人、朋友、同事的照片都能快速找到。

- 自动生成每个人的头像
- 可自定义人物名称
- 人物数据只保存在本地

### 📅 时间线浏览
按拍摄时间自动整理照片，按年、月、日分组，轻松找回某段回忆。

- 自动读取 EXIF 拍摄时间
- 支持从文件名识别时间
- 可查看拍摄设备、地点等信息

### 🔒 完全本地，隐私优先
- **不上传照片**：所有照片、索引、人物数据都保存在你的电脑上
- **可离线使用**：首次下载 AI 模型后，无需联网也能正常搜索
- **数据透明**：处理方式完全透明，你的照片始终由你掌控

## 技术架构与隐私保障

觅影采用**本地优先**架构，所有 AI 计算、索引生成和数据存储都在你的电脑上完成，不需要把照片交给任何第三方服务器。

| 层级 | 技术选型 | 隐私意义 |
|------|----------|----------|
| 前端界面 | Vue 3 + TypeScript + Vite | 仅在本地运行，不嵌入任何追踪脚本 |
| 桌面框架 | Tauri 2.11 + Rust | 原生性能，较小的攻击面，无后台服务器通信 |
| 语义搜索 | Chinese-CLIP（ONNX） | 模型在本地加载推理，照片描述不会上传 |
| 人脸识别 | YuNet（人脸检测）+ MobileFaceNet（人脸识别），ONNX 格式 | 人脸检测与特征提取均在本地完成，不上传云端 |
| 向量数据库 | LanceDB | 照片语义索引和人脸特征库均存储在本地 |
| 数据目录 | `%APPDATA%/seekphoto/` | 索引、缩略图、模型、人脸数据默认保存在你本机，可随时管理或删除 |

此外，觅影**无需注册账号、不收集使用数据、不连接私有服务器**。你的照片、人物分组和搜索记录始终只属于你。
